*Chapter 29 · 제29장*
# Korean Spelling Rules
**한국어 맞춤법 — 한국인도 헷갈리는 규칙**

맞춤법 · maj-chum-beop · The Rules Native Koreans Struggle With

💡 **Tip:** Even native Koreans struggle with 맞춤법! A single wrong letter completely changes the meaning. Koreans judge each other's writing by spelling — get it right and sound educated! 🎓

## 🔥 #1 Most Common Mistake — 되 vs 돼

This is the #1 spelling error even Koreans make. Easy rule below.

> **되** (doe) = verb stem of 되다 (to become)
> **돼** (dwae) = contracted form of 되어 (becoming / becomes)

> 🎯 **Test trick:** Replace 되/돼 with "해" or "하":
> - If "해" sounds right → use **돼**
> - If "하" sounds right → use **되**

| Wrong | Right | English |
|---|---|---|
| 안돼요 ✗ | 안 돼요 ✓ | not okay |
| 됬어요 ✗ | 됐어요 ✓ | it's done |
| 안되 ✗ | 안 돼 ✓ | no / not allowed |
| 어떻게 되 ? ✗ | 어떻게 돼? ✓ | how does it become? |
| 잘 됬어 ✗ | 잘 됐어 ✓ | it went well |

## 🔥 #2 — 안 vs 않

| Korean | Function | Usage |
|---|---|---|
| **안** | adverb (before verb) | 안 가요 (don't go) |
| **않** | auxiliary verb (with -지) | 가지 않아요 (don't go) |

> 🎯 **Test trick:** Can you put it before the verb directly?
> - YES → use **안**
> - NO (needs -지) → use **않**

| Wrong | Right |
|---|---|
| 안하다 ✗ | 안 하다 ✓ (separate words!) |
| 가지 안다 ✗ | 가지 않다 ✓ |
| 먹지 안아요 ✗ | 먹지 않아요 ✓ |

## 🔥 #3 — 에 vs 의 vs 이

These three look similar but have different roles.

| Korean | Pronunciation | Function |
|---|---|---|
| 에 | e | location / time marker |
| 의 | ui (often pronounced 에) | possessive ("'s") |
| 이 | i | subject marker (after consonant) |

| Example | Translation |
|---|---|
| 학교**에** 가요 | I go to school (에 = direction) |
| 친구**의** 책 | My friend's book (의 = possessive) |
| 책**이** 있어요 | There is a book (이 = subject) |

## 🔥 #4 — Double Consonant Spelling

Korean has tense (쌍) consonants that change meaning entirely.

| Plain | Tense | Example pair |
|---|---|---|
| ㄱ | ㄲ | 가다 (go) vs 까다 (peel) |
| ㄷ | ㄸ | 달 (moon) vs 딸 (daughter) |
| ㅂ | ㅃ | 불 (fire) vs 뿔 (horn) |
| ㅅ | ㅆ | 사다 (buy) vs 싸다 (cheap) |
| ㅈ | ㅉ | 자다 (sleep) vs 짜다 (salty) |

## 🔥 #5 — Common Spacing Errors

Korean spacing (띄어쓰기) is notoriously difficult. Top rules:

| Rule | Example |
|---|---|
| Verb + 수 있다/없다 → space | 갈 수 있어요 (NOT 갈수있어요) |
| Number + counter → space | 두 명 (NOT 두명) |
| 같이 vs 가치 | 같이 = together, 가치 = value |
| 안 + verb → space | 안 해요 (NOT 안해요) |

## 🔥 #6 — Verbs with Confusing Spelling

### 잘 vs 잘 ←→ 잘하다 / 잘 하다

| Korean | Meaning |
|---|---|
| 잘하다 | to do well (one word, no space) |
| 잘 하다 | to do correctly (two words, with space) |

### 못 vs 못하다

| Korean | Meaning |
|---|---|
| 못하다 | cannot do (one word) |
| 못 하다 | failed to do (two words) |

## 🔥 #7 — Final Consonant Changes in Writing

| Pronounced | Actually written | Example |
|---|---|---|
| 일거요 | 읽어요 | I read |
| 안자요 | 앉아요 | I sit |
| 올타 | 옳다 | correct |
| 흐려저요 | 흐려져요 | becomes cloudy |

When in doubt, check the dictionary form — never write phonetically.
